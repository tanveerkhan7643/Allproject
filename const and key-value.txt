Object ek collection of keyâ€“value pairs hota hai.

Key = naam (string type hota hai)

Value = koi bhi data (string, number, boolean, array, ya dusra object)

Matlab object ek aisa data structure hai jisme tum data ko property ke form me store karte ho.

Example:
// Object ka example
const person = {
  name: "Ali",   // yaha 'name' ek key hai aur "Ali" uski value hai
  age: 22,       // 'age' key hai aur 22 value hai
  isStudent: true
};

console.log(person.name); // Ali
console.log(person.age);  // 22
